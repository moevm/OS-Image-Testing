## Сборка Yocto образа через докер

Скрипт направлен на базовую разовую инициализацию тома докера и последующую многоразовую сборку образа.

Для начала необходимо запустить скрипт `handler.sh`, с аргументом `-b` или `--build` для установки среды и сборки образа в том.

``
./handler --build
``

После этого можно произвести запуск с аргументом `-r` или `--run` для запуска qemu.

``
./handler --run
``

Для добавления теста с использованием, например **stress-ng**, нужно добавить тест в папку tests и обновить соответствующий .bb файл.

Для добавления новой утилиты нужно обновить local.conf и прописать соответствующий рецепт, затем добавить в handler.
