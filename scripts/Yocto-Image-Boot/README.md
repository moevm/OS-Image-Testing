## Сборка Yocto образа через докер

Скрипт направлен на базовую разовую инициализацию тома докера и последующую многоразовую сборку образа.

Для начала необходимо запустить скрипт `handler.sh`, с аргументом `build` для установки среды и сборки образа в том.

``
./handler build
``

После этого можно произвести запуск с аргументом `run` для запуска qemu.

``
./handler run
``

Для тестирования добавлен аргумент `test`, сами тесты нужно прописать.

``
./handler test
``
