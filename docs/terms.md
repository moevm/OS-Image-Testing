## Основные термины

*Временной ряд* — упорядоченная последовательность точек вида (value, timestamp), относящихся к одной и той же метрике с фиксированным набором меток. Любое изменение значения любой метки образует уже другой временной ряд. [1]

*Системное тестирование* — тестирование, выполняемое на полной, интегрированной системе для оценки того, удовлетворяет ли система установленным требованиям. [2]

*Мониторинг* — постоянная проверка, надзор, критическое наблюдение или определение статуса с целью выявления отклонений от требуемого или ожидаемого уровня производительности. [3]

*timestamp* — однозначное значение временного кода, зарегистрированное для конкретного события с использованием заданного источника времени. [4]

*Метрики* — количественные меры степени, в которой система, компонент или процесс обладают заданным атрибутом. [5]

*Статистики* — совокупность метрических данных, агрегированных за определённые периоды времени. [6]

*Профилирование* — метод, используемый при разработке программного обеспечения для измерения и анализа поведения программы во время выполнения. [7]

*Трассировка* — инструмент наблюдения, который фиксирует путь, пройденный запросом по мере его распространения по распределённой системе или набору сервисов. [8]

*Задержка* — интервал времени, начинающийся с момента, когда последний бит входного кадра достигает входного порта устройства, и заканчивающийся в момент, когда первый бит соответствующего выходного кадра появляется на выходном порту. [9]

*Пропускная способность* — максимальная скорость, при которой устройство обрабатывает все поданные кадры без потерь (то есть ни один кадр не отбрасывается). [9]


## Сокращения

*BIOS (Basic Input/Output System)* — микропрограмма, встроенная в материнскую плату, которая запускается до операционной системы, инициализирует и проверяет оборудование и предоставляет базовые инструкции для загрузки ОС. [10]

*UEFI (Unified Extensible Firmware Interface)* — спецификация, определяющая модель интерфейса между операционными системами и встроенным программным обеспечением платформы. [11]

*LTP (Linux Test Project)* — обширный набор тестов для тестирования ядра Linux и связанных с ним функций с целью проверки стабильности и надёжности. [12]

*PTS (Phoronix Test Suite)* — кроссплатформенный фреймворк для автоматизированного бенчмаркинга. [13]

*RRD (Round Robin Database)* — база данных фиксированного размера для хранения данных временных рядов с использованием кольцевого буфера, в котором старые данные перезаписываются по мере добавления новых. [14]
