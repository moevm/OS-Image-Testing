# Основные термины

*[Временной ряд](https://victoriametrics.com/blog/the-rise-of-open-source-time-series-databases/)* — это упорядоченная последовательность точек (value, timestamp), относящихся к одной и той же метрике с фиксированным набором меток. Любое изменение значения любой метки образует уже другой временной ряд.

*[Системное тестирование](https://www.iso.org/obp/ui/en/#iso:std:iso-iec-ieee:24765:ed-2:v1:en)* — тестирование, проводимое на полной, интегрированной системе для оценки соответствия системы установленным требованиям.

*[Мониторинг](https://www.iso.org/obp/ui/#iso:std:iso:guide:73:ed-1:v1:en)* — постоянная проверка, надзор, критическое наблюдение или определение статуса с целью выявления отклонений от требуемого или ожидаемого уровня производительности.

*[timestamp](https://www.itu.int/dms_pubrec/itu-r/rec/tf/r-rec-tf.686-3-201312-i!!pdf-e.pdf)* — однозначное значение временного кода, зарегистрированное для конкретного события с использованием заданного источника времени.

*[Метрики](https://www.informatik.htw-dresden.de/~hauptman/SEI/IEEE_Standard_Glossary_of_Software_Engineering_Terminology%20.pdf)* — количественная мера степени, в которой система, компонент или процесс обладают заданным атрибутом.


*[Статистики](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/cloudwatch_concepts.html)* — совокупность метрических данных за определенные периоды времени.


*[Профилирование](https://grafana.com/docs/pyroscope/latest/introduction/what-is-profiling/)* —  метод, используемый при разработке программного обеспечения для измерения и анализа поведения программы во время выполнения.


*[Трассировка](https://www.prisma.io/blog/tracing-tutorial-prisma-pmkddgq1lm2#what-is-tracing)* — это инструмент наблюдения, который записывает путь, пройденный запросом по мере его распространения по вашим приложениям.



## Сокращения

[BIOS (Basic Input/Output System)](https://www.crucial.com/articles/pc-builders/bios-vs-uefi) — микропрограмма, встроенная в материнскую плату, которая запускается до операционной системы, инициализирует и проверяет оборудование и предоставляет базовые инструкции для загрузки ОС.

[UEFI (Unified Extensible Firmware Interface)](https://uefi.org/faq) — спецификация, определяющая новую модель интерфейса между операционными системами персональных компьютеров и встроенным ПО платформы.


[LTP (Linux Test Project)](https://linux-test-project.readthedocs.io/) — обширный набор тестов для тестирования ядра Linux и связанных с ним функций для проверки стабильности и надёжности.
  

[PTS (Phoronix Test Suite)](https://www.phoronix-test-suite.com/) — кроссплатформенный фреймворк для автоматизированного бенчмаркинга.


[RRD (Round Robin Database)](https://oss.oetiker.ch/rrdtool/) —
  база данных фиксированного размера, которая хранит данные временных рядов с использованием кольцевого буфера, перезаписывая старые данные по мере добавления новых.
